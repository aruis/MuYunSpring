# 导入导出

导入导出面向动态模块交换能力，提供模板、解析、执行、错误文件和数据导出闭环。当前只承诺开启 `EXCHANGE` 能力的动态主实体。

## 能力范围

- 模板可根据已发布的动态模块字段与关系配置生成工作簿。
- 引用字段可生成辅助下拉，也可按字段关闭。
- 导入支持引用 title 反推。
- 导出支持引用 title 输出。
- 导出支持全量查询上下文和选中记录两种入口。
- 支持一层主子结构的子表导出。
- 导入错误可生成错误文件，并通过 token 下载。
- Web 暴露面按 `EXCHANGE` 能力门控。

## 业务使用口径

模板归属于导入能力，用于指导用户准备导入文件。模板中的引用下拉只是辅助输入；即使关闭下拉，导入仍应支持 title 反推。

导入执行区分解析和写入：解析用于提前识别文件结构与数据问题，执行才进入动态记录写链路。执行结果返回新增、更新、跳过、错误数量和错误文件信息。

导出复用动态查询上下文，可接收分页、排序、查询模板、外部查询值、快速搜索和手工条件。选中导出只是追加记录 ID 条件，仍走标准导出动作、数据范围和租户链路。

用户级导出字段、列宽、排序等个人化设置优先由页面偏好承载；只有出现可共享、可发布的系统级导出模板时，才单独抽象模板模型。

## 后续增强

以下能力不属于当前收口范围：引用 projection、跨 sheet 辅助页、静态业务导入导出、更复杂子表层级和 dry-run 后的配置草稿应用。
